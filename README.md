# Nesta atividade foram realizadas os seguintes requisitos:


- A equipe de Interface Web iniciou o projeto Django e estruturou as páginas iniciais do sistema, incluindo telas de login, cadastro e página inicial. Foi implementado um sistema de autenticação simples utilizando Django e -SQLite para controle básico de acesso. O site encontra-se funcional e acessível via navegador, atendendo aos requisitos mínimos da etapa.

- criação de wireframes simples em HTML para as telas de Login, Cadastro e Home. As telas de Login e Cadastro apresentam formulários básicos de autenticação, enquanto a tela Home exibe a mensagem de boas-vindas e opções de navegação do sistema, como acesso às notícias e logout.

- Foi implementada uma página de listagem de notícias utilizando dados simulados para testes de interface. Também foi aplicado controle básico de sessão no login, permitindo acesso às páginas apenas para usuários autenticados, atendendo aos requisitos da etapa.
